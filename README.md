# Smartfinance-AI
AI-powered personal finance management platform for expense tracking, budget planning, savings analysis, and smart financial insights using React, Node.js, MongoDB, Docker, and AWS.
SmartFinance AI is a modern AI-powered personal finance management platform designed to help users understand, track, and improve their financial health.

The platform enables users to securely manage income and expenses, upload bank statements, visualize spending through interactive dashboards, and receive AI-powered financial recommendations for smarter budgeting and saving.

Built using a modern full-stack architecture, SmartFinance AI combines an intuitive React frontend with a scalable Node.js and Express backend, MongoDB Atlas for cloud database storage, and AWS & DevOps technologies for deployment and automation.

The goal of this project is to demonstrate real-world Full Stack Development, Cloud Computing, and DevOps skills while solving a practical financial management problem.

##  Features

- 🔐 Secure User Authentication
- 💳 Income & Expense Tracking
- 📊 Interactive Financial Dashboard
- 📈 Spending Analytics & Reports
- 📄 Bank Statement PDF Upload
- 🤖 AI-Powered Financial Insights
- 💰 Budget Planning & Savings Goals
- 📱 Fully Responsive UI
- ☁️ Cloud Deployment on AWS
- 🐳 Dockerized Application
- 🚀 CI/CD using GitHub Actions
